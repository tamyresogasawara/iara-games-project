# Iara Games - Versão 2.0

Este é o repositório da versão 2.0 do projeto Iara Games, uma plataforma dedicada a promover e vender jogos de estúdios indies brasileiros.

## Descrição do Projeto

A Iara Games v2 é uma evolução da versão inicial, com as seguintes melhorias:

- **Estrutura de Pastas Corrigida:** A organização dos arquivos foi refeita para garantir que o CSS e as imagens carreguem corretamente.
- **Página de Cadastro:** Uma nova página de formulário foi adicionada para permitir o cadastro de novos usuários.
- **Acessibilidade Aprimorada:** Ambas as páginas foram desenvolvidas com foco em acessibilidade, seguindo as diretrizes do WCAG 2.1.
