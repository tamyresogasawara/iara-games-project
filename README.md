# Iara Games - Versão 2.0

Este é o repositório da versão 2.0 do projeto Iara Games, uma plataforma dedicada a promover e vender jogos de estúdios indies brasileiros.

## Descrição do Projeto

A Iara Games v2 é uma evolução da versão inicial, com as seguintes melhorias:

- **Estrutura de Pastas Corrigida:** A organização dos arquivos foi refeita para garantir que o CSS e as imagens carreguem corretamente.
- **Página de Cadastro:** Uma nova página de formulário foi adicionada para permitir o cadastro de novos usuários.
- **Acessibilidade Aprimorada:** Ambas as páginas foram desenvolvidas com foco em acessibilidade, seguindo as diretrizes do WCAG 2.1.
- **Logo e Identidade Visual:** Logo da Iara Games adicionado ao header de todas as páginas, com design responsivo que se adapta a diferentes tamanhos de tela.
- **Favicon:** Favicon personalizado adicionado para melhorar a identidade visual da plataforma nas abas do navegador.
- **Página "Sobre Nós":** Nova página dedicada apresentando a missão, valores e objetivos da Iara Games, incluindo informações sobre curadoria, comunidade e colaboração.
- **Tipografia:** Fonte Inter implementada em todo o site para uma experiência visual moderna e consistente.
- **Design System:** Botões CTA (Call-to-Action) centralizados por padrão para melhor usabilidade.

## Estrutura do Projeto

### Páginas Disponíveis

- **`index.html`** - Página inicial com apresentação dos jogos em destaque, seção sobre a plataforma e informações para desenvolvedores
- **`pages/cadastro.html`** - Página de cadastro de usuários com formulário completo e validação
- **`pages/sobre.html`** - Página "Sobre Nós" com informações detalhadas sobre a missão, curadoria e comunidade da Iara Games

### Arquivos de Recursos

- **Logo:** `images/logo iara games.png` - Logo principal da plataforma
- **Favicon:** `images/iara fav icon.png` - Ícone exibido nas abas do navegador
- **CSS:** 
  - `css/styles.css` - Estilos principais do site
  - `css/form-styles.css` - Estilos específicos para formulários
- **JavaScript:**
  - `js/main.js` - Funcionalidades principais
  - `js/form-validation.js` - Validação de formulários
